mongodb数据库

发表于 2017-11-20 | 分类于 数据库
字数统计: 390 | 阅读时长 ≈ 2

mongod —dbpath 路径开启数据库，如果数据库不存在则会创建

mongo连接数据库

show dbs 看一下目前有几个数据库，默认是admin和local两个数据库

use 名字 可以新建一个数据库，但是如果这个数据库已经存在了，就会进入到这个数据库，例如use test，里面必须有内容，才会在show dbs显示出来

show collections

必须使用use test连接之后，db.user.insert({‘’:’’,’’:’’})

db.user.find()可以查出现在里面的内容是什么

总结一下

mongoimport —db test —collection user —drop —file dir

—db导入到哪个库

—collection导入到哪个集合

—drop如果加上就表示将原有的数据清空

—file 要导入的文件

关于查询的操作

db.name.find() //查询所有文档

db.name.find({‘key’:’value’})

db.name.find({‘key1’:’value1’,’key2’:’value2’})

db.name.find({$or:[{‘key1’:’value1’},{‘key2’:’value2’}]})

db.name.find(k:{$lt: v})

db.name.find(k:{$gt: v})

db.name.update(
{k:v},
{
$set:{k2:v2,k3:v3}
}
)

db.name.update(
{k:v},
{
k2:v2,
k3:v3
},
{multi:true}
)
db.name.remove({k:v})//删除所有匹配到的文档

db.name.remove({k:v},{justone:true})//删除第一个匹配的文档

db.name.remove({})//删除全部集合

db.name.drop()//删除集合

db.dropDatebase() //删除当前所在的数据库
