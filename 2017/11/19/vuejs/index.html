学习了这么多vue的教程，实际来做一个项目还是不知道如何上手，因此，我根据网上的饿了么实战视频教程来总结一下用到的知识，梳理一下我的思路。

首先

vue的脚手架帮助我们编写好一个基础的代码，使用到了vue-cli工具。

npm install -g vue-cli 可以全局的安装vue命令，只需要安装一次即可。

vue init <template-name> <project-name> 是初始化一个项目，可以官方从事先设置好的项目里安装，也可以自定义的从自己的仓库里面安装，例如：vue init webpack myDemo便生成了一个myDemo的目录，以webpack打包工具为基础

cd myDemo 进入到myDemo文件里面

npm install 安装项目运行所需要的依赖

npm run dev 默认在8080端口就跑起来了你的项目

项目文件目录

build和config目录是有关于webpack的配置，node_modules目录是我们通过npm install安装的依赖代码库，src目录是我们用来存放项目的源码，也就是主要的工作区域，static目录是用来存放第三方静态资源。

.babelrc是专门处理ES6的语法

.editorconfig是对编辑器的设置

.eslintignore用来忽略语法检查的目录文件

.eslintlrc.js是eslint的配置文件

.gitignore是git提交时会忽略掉的一些文件

index.html是入口的html文件

package.json项目的配置文件

README.md项目的描述文件

cli.im草料二维码
