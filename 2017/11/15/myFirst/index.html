字数统计: 1,569 | 阅读时长 ≈ 6

最近学习数学分析的时候，发现在多重积分这一章节里面有好多涉及到解析几何的问题，比如空间曲线的方程，现在总结一下，方便以后记忆。

柱面、锥面、旋转曲面与二次曲面

1.柱面

定义：在空间，由平行于定方向且与一条定曲线相交的一族平行直线所生成的曲面叫做柱面，定方向叫做柱面的方向，定曲线叫做柱面的准线，那组平行直线中的每一条直线，都叫做柱面的母线。

那么，如何判断一个面是不是柱面，我们有以下定理:

定理1：在空间直角坐标系中，只含两个元（坐标）的三元方程是一个柱面，它的母线平行于所缺元（坐标）的同名坐标轴。

说白了，就是看似三元方程，实际上只有两元。例如F(x, y) = 0, G(y, z) = 0, H(x, z) = 0, 都表示柱面，他们的母线分别平行于z轴、x轴、y轴。

因此，根据这个定理，我们可以得到

$\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$

$\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$

$y^2 = 2px$

这些都代表了柱面。上面的三个公式我们在高中就已经熟悉，但是那只是在平面上来研究，如果我们放在空间直角坐标系里，上面的柱面与xOy坐标面的交线分别是椭圆、双曲线与抛物线。所以他们依次叫做椭圆柱面、双曲柱面、抛物柱面，它们的方程都是二次的，所以统称为二次柱面。

2.锥面

定义：在空间通过一定点且与定曲线相交的一族直线所生成的曲面叫做锥面，这些直线都叫做锥面的母线，那个定点叫做锥面的顶点，定曲线叫做锥面的准线。

同样的，我们如何判断一个锥面呢，有以下定理：

定理2：一个关于x, y, z的齐次方程总表示顶点在坐标原点的锥面。

这里我们要注意:

  1. 柱面和锥面都有准线和母线，柱面有方向，而锥面有顶点。
  2. 关于齐次方程，课本上给的定义是：设 $\lambda$为实数，对于函数f(x, y, z)，如果有f(tx, ty, tz) = $t^\lambda$f(x, y, z)，其中t的取值应当使$t^\lambda$有确定的意义，那么f(x, y, z)叫做 $\lambda$次齐次函数，f(x, y, z) = 0叫做 $\lambda$次齐次方程。这个定义可以推广到n个变量的情况。我们可以看到，齐次函数就是如果我们把函数的每一个变量乘以一个同一个因子的话，其函数值变为原函数乘以这个因子的 $\lambda$次幂。
  3. 不难理解，关于x - $x_0$, $y - y_0$, $z - z_0$的齐次方程表示顶点在($x_0$, $y_0$, $z_0$)的锥面。
  4. 特别的，关于x, y, z的齐次方程可能只表示一个原点。即除了原点之外，曲面上再也没有其他的实点，例如$x^2 + y^2 + z^2 = 0$这样的曲面，我们把它叫做具有实顶点的虚锥面。

3.旋转曲面

定义：在空间，一条曲线 $\Gamma$绕着定直线l旋转一周所生成的曲面叫做旋转曲面，或称回转曲面，曲线 $\Gamma$叫做旋转曲面的母线，定直线l叫做旋转曲面的旋转轴，简称为轴。

对于旋转曲面上的任意一点 $M_1$在旋转是形成一个圆，这个圆也就是通过点 $M_1$且垂直于轴l的平面与旋转曲面的交线，叫做纬圆或称纬线。在通过旋转轴l的平面上，以l为边界的任意半曲面都与曲面交成一条曲线，这些曲线在旋转中显然都能彼此重合，这条曲线叫做旋转面的经线。

特别的，将母线所在平面取坐标面，将旋转轴作为坐标轴，有以下规律。

当坐标平面上的曲线 $\Gamma$绕此坐标平面的一个坐标轴旋转时，只需要将曲线在坐标面里的方程保留和旋转轴同名的坐标轴，而以其他两个坐标平方和的平方根来代替另一坐标。

zoy平面里的曲线绕y轴旋转：

$F(y, \pm \sqrt{x^2 + z^2}) = 0$

zoy平面里的曲线绕z轴旋转：

$F(\pm \sqrt{x^2 + y^2}, z) = 0$

下面写出几种旋转曲面的方程———

长形旋转椭球面：$\frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{b^2} = 1$

扁形旋转椭球面：$\frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{a^2} = 1$

单叶旋转双曲面：$\frac{x^2}{b^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = 1$

双叶旋转双曲面：$\frac{y^2}{b^2} - \frac{x^2}{c^2} - \frac{z^2}{c^2} = 1$

旋转抛物面：$x^2 + y^2 = 2pz$

4.椭球面

椭球面的标准方程：$\frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1$，其中a, b, c为任意的正常数，通常假定a > b > c。

单叶双曲面标准方程：$\frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{z^2}{c^2} = 1$，其中a, b, c为任意的正常数。

注意：$\frac{x^2}{a^2} - \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1$和$ - \frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1$

双叶双曲面的标准方程：$\frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{a^2} = -1$，其中a, b, c为任意的正常数。

注意：$\frac{x^2}{a^2} - \frac{y^2}{b^2} + \frac{z^2}{c^2} = -1$和$-\frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{c^2} = -1$

椭圆抛物面的标准方程：$\frac{x^2}{a^2} + \frac{y^2}{b^2} = 2z$，其中a, b是任意的正常数。

双曲抛物面的标准方程：$\frac{x^2}{a^2} - \frac{y^2}{b^2} = 2z$，其中a, b是任意的正常数。

椭圆抛物面和双曲抛物面统称为抛物面，它们都没有对称中心，因此又叫做无心二次曲面。
