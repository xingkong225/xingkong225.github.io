Matlab学习笔记(一)

发表于 2018-03-10 | 分类于 专业课
字数统计: 1,893 | 阅读时长 ≈ 8

本着越简单越好的原则，我会把自己看到的内容用自己的话描述一下，由于我在绘图这一方面所知的有所欠缺，因此先整理一下这方面的笔记，我默认读者已经知道了一些基础的知识，那么，我们就开始吧。

基本二维绘图函数有：

  1. plot()函数

    plot(Y)

    若Y为向量，则以向量的索引值为横坐标，以向量的每一个分量作为纵坐标画图；

    若Y为实数矩阵，则将每列看做是一个向量，即有多少列就有多少函数的图像。以每列的值为纵坐标，以每列的值的索引为横坐标画在同一个图中。

    若Y为复数矩阵，则将Y的实部和虚部分开，看做两个矩阵，即为plot(real(Y),imag(Y))，关于两个分量的画图，见下一个函数。

    plot(X,Y)

    最常见的是画出函数的图形，若X和Y均为一维向量，则建立起Y对应于X的图像，即以Y为纵坐标，以X为横坐标，此时要求X和Y中所含分量的个数必须一致。

    例如： x = linspace(-2*pi, 2*pi); y = 3 * sin(x); plot(x,y);

    若X和Y均为矩阵，则要求X和Y的大小必须一致，以Y每列的数为纵坐标，对应于X的每一列的元素，个人感觉用的很少。

    plot(X,Y,lineSpec)

    我们可以在画图时指定图像的线型，颜色和标记符号。

    颜色：b — 蓝色；g — 绿色；r — 红色；c — 青色；m — 品红色；k — 黑色；y — 黄色；w — 白色。

    线型：实线 ‘-‘ 虚线 ‘—‘ 点线 ‘:’ 点划线 ‘-.’。

    标记符号：点号 ‘.’ 空心圆 ‘o’ 叉号 ‘x’ 加号 ‘+’ 星号 ‘*’ 下三角形 ‘v’ 上三角形 ‘^’ 左三角形 ‘<’ 右三角形 ‘>’ 正方形 ‘s’ 菱形 ‘d’ 五角星形 ‘p’

    这些不需要死记硬背，都是很形象的符号。

    当然了，也有对曲线属性的一些设置，比如线条宽度LineWidth,标记颜色MarkerFaceColor,标记边框颜色MarkerEdgeColor,标记大小MarkerSize等等，我感觉大部分情况下用不上，具体的可以去Google一下。

    plot(x,y,'b:s','LineWidth',4,'MarkerFaceColor','r','MarkerSize',10,'MarkerEdgeColor','g');

    另外，plot()函数还可以同时画多个函数的图像，例如：x = linspace(0,2*pi);y1 = sin(x)./x;y2 = cos(x)./x;plot(x,y1,'b-',x,y2,'r:');

  2. plotyy()函数

    plotyy(X1,Y1,X2,Y2)

    双坐标轴绘图，在同一个坐标轴中画出两个函数的图像，即X1，Y1和X2，Y2。并且[AX,H1,H2] = plotyy(…)是这个函数的返回值，AX是坐标轴句柄，H1和H2分别是两个函数句柄。(通过句柄我们就可以操纵相应的对象)

  3. fplot()函数

    fplot(function,limit,lineSpec)

    在我们画图时，很有可能提前不知道函数是怎么样随着自变量变化的，有时候画出来的图形可能达不到我们想要的效果。简单的说，通过fplot()函数中的自适应算法，在画图时，在函数图像密集的地方自变量取值间隔小，在函数图像平坦的地方函自变量取值间隔大，这样就很方便的保证了画图的质量。

    其中function如果直接写成函数fplot('x.^2',[2,4])，matlab会提出警告。因此此处应指的是匿名函数即fun = @(arg1,arg2,…) expr，arg是参数，expr可以是一个表达式，也可以是两个表达式，需要用[ ]，两个表达式中间用,隔开。例如 fun = @(x) [2*cos(x)./x,x.^2];fplot(fun,[-20,20])，便可以画出在区间2~4上两个函数图像。

    limit可以是二维向量[xmin,xmax]，也可以是四维向量[xmin,xmax,ymin,ymax]

    lineSpec是指定函数的线型，颜色和标记符号。

    注意输入的函数必须为单一变量。

  4. ezplot()函数

    ezplot(function,limit)

    ezplot()函数可以直接绘制一元函数、参数方程式的函数以及隐函数的图形。

    ezplot(function,[min,max])：绘制函数在min< x <max 上的图形，默认为-2*pi< x <2*pi。并且可以指定y的取值范围，ezplot(function,[xmin,xmax,ymin,ymax])。

    ezplot(f)，其中f = f(x,y)，绘制隐函数f(x,y) = 0在-2*pi< x,y <2*pi上的图形。同样的可以指定x和y的范围。

    ezplot(x,y,[tmin,tmax])，可以绘制x = x(t),y = y(t)在tmin< t < tmax上的图形，默认为0< t < 2*pi。

    例如：绘制圆 $x^2 + y^2 = R^2$的图形 参数形式ezplot(@(t) cos(t), @(t) sin(t),[0,2*pi]); 隐函数形式 ezplot('x^2 + y^2 - 1',[-sqrt(1),sqrt(1)]);
    `

  5. polar()函数

    polar(theta,r,lineSpec)

    前面说的都是一些常规的直角坐标的图形，我们当然可以在matlab中画出由极坐标或者柱坐标表示的图形。

    例如著名的阿基米德螺旋线 r = a + b*$\theta$

    theta = linspace(0,6*pi,500);r = 1 + 2*theta;polar(theta,r,'b-');

三维图形的绘制有着更加丰富的函数，让我们来试试吧。

  1. 三维基本绘图函数：plot3()

    plot3(x,y,z,lineSpec…)

    这个…就不说了吧，主要是用于绘制三维的曲线图形，没有网格，没有填充色。

  2. 绘制网格图形：mesh(),meshc(),meshz()

    mesh(X,Y,Z)

    mesh()用来创建普通的网格图，meshc()用来创建带有等高线的网格图，meshz()用来创建带有围裙的网格图，他们的调用格式完全一样。

    在使用mesh函数时，最常用的就是meshgrid()函数生成网格数据。

    简单来说，[X,Y] = meshgrid(x,y) 其中x和y都是一维向量，生成的X是x按照y元素的个数的行复制，生成的Y是y按照x元素的个数的列复制。例如a = [3,4,5], b = [1,2],则[X,Y] = meshgrid(a,b)得到的X是[3,4,5;3,4,5],Y是[1,1,1;2,2,2],可能这样说还是不是很直观，我们可以拿起笔在纸上按照正常写矩阵的方式把它们写出来，自然就一目了然了。值得一提的是，meshgrid(x)与meshgrid(x,x)等同。

  3. 绘制曲面图形(也就是上了颜色的)：surf(),surfc(),surfl()

    surf(X,Y,Z)

    surf()用来创建基本的着色图形，surfc()用来创建带有等高线的曲面图，surfl()可以控制光照效果。前两个用法与mesh()函数较为类似。

  4. 绘制隐函数三维图形：ezmesh()

    ezmesh(f,limit)

    此时f = f(x,y)，可以绘制出f(x,y) = 0的图形，limit是x和y的取值范围，即可以是二维向量[min,max]表示min< x,y <max,也可以是四维向量[xmin,xmax,ymin,ymax]，默认是-2*pi< x,y <2*pi。

    ezmesh(x,y,z,limit)

    此时曲面使用参数方程表示出来，即x = x(u,v),y = y(u,v),z = z(u,v)，limit是u和v的取值范围，默认是[-2*pi< u <2*pi,-2*pi< v <2*pi]。也可以自己设定[umin,umax,vmin,vmax]

值得一提的是，ezmesh()和mesh()两个函数最主要的区别是，mesh()函数传入的参数是一个一个数值矩阵，但是ezmesh()函数的参数是隐函数或者是以参数方程表示的函数。例如要想画出来$x^2 + y^2 = 1$函数的图像，我们可以使用mesh()函数：x = [-2:0.2:2]; y = [-3:0.2:3]; [X,Y] = meshgrid(x,y); Z = X.^2 + Y.^2 -1; mesh(X,Y,Z);，也可以使用ezmesh()函数：fun = @(x,y) x.^2 + y.^2 - 1; ezmesh(fun);
