随机向量

在定义随机向量之前，我们先来回顾一下随机变量的数学定义。如果$(\Omega,\mathcal{F},P)$是一个概率空间，$X(\omega)$是定义在$\Omega$上并且取值为一条直线的函数，并且这个函数小于等于任何一个数是一个事件，而且这个事件是在$\mathcal{F}$中，并且$P$是$\mathcal{F}$上的一个函数，叫做概率。

定义：设$X_1, X_2, \cdots,X_k,(k \ge 1)$是$k$个定义在同一个概率空间$(\Omega,\mathcal{F},P)$上的随机变量，则称$\vec{X}=(X_1, X_2, \cdots,X_k)$是$(\Omega,\mathcal{F},P)$上的随机向量。我们称$F_{X_1, X_2, \cdots,X_k}(x_1, x_2, \cdots,x_k)=P(X_1 \leq x_1, X_2 \leq x_2,\cdots,X_k \leq x_k)$为随机向量的联合分布函数(或者也可以说是分布函数)

二维分布函数的性质：

(1) $F(-\infty,y)=\lim\limits_{x \rightarrow -\infty}F(x,y)=0, \forall y$

$F(x,-\infty)=\lim\limits_{y \rightarrow -\infty}F(x,y)=0, \forall x$

$\lim\limits_{x \rightarrow \infty,y \rightarrow \infty}F(x,y)=F(\infty,\infty)=1$

(2) 如果$x_1 < x_2, y_1 < y_2$，则有

$P(x_1 < X \leq x_2;y_1 < Y \leq y_2)=F(x_2,y_2)-F(x_2,y_1)-F(x_1,y_2)+F(x_1,y_1) \ge 0$

(3) $F(x,y)$对于每一个参数都是右连续，即$\lim\limits_{h \rightarrow 0+}F(x+h,y)=\lim\limits_{h \rightarrow 0+}F(x,y+h)=F(x,y)$

任何满足上述三条性质的函数都叫做二维分布函数，这样做的好处是摆脱了随机变量的定义。

定义：如果$F_{X,Y}(x,y)$是二维联合分布函数，则称$F_X(x),F_Y(y)$为边缘分布函数。

显然，有$F_X(x)=F_{X,Y}(x,\infty), F_Y(y)=F_{X,Y}(\infty,y)$

同时也有一个不等式：$F_X(x)+F_Y(y)-1 \leq F_{X,Y}(x,y) \leq \sqrt{F_X(x)F_Y(y)}$，对于任意的$x,y$

注意此时$\vec{X}$是以$\Omega$为定义域，取值在$n$维欧式空间中的一个函数，这是一个从一元到多元函数的一个转变。此时$X_i, 1 \leq i \leq n$都是定义在同一个概率空间中并且都是以$\Omega$为定义域的。

那么我们为什么要研究多个随机变量组成的随机向量呢？因为有些随机变量之间可能是有关系的，如果我们单独研究某一个随机变量，我们可能得不到一个整体的结构，也就是随机向量所含的信息要比单个的随机变量$X_i, 1 \leq i \leq n$所含的信息多得多，如果我们知道了随机向量的分布，我们从联合分布函数就可以推出每一个随机变量的边缘分布。

例如我们要想知道$X_1$的分布如何，只需在随机向量的联合分布函数中令除$X_1$外其他的随机变量的值取$+\infty$即可。即$F_1(x)=P(X_1 \leq x_1)=F(x_1,\infty,\infty,\cdots,\infty)$，不仅如此，我们还可以求得由$\vec{X}$每一个分量的不同组合得到一个子向量的分布，只需令没有被提到的随机变量对应位置写成$\infty$即可，由此我们可以得到$C_n^1+C_n^2+\cdots+C_n^{n-1}=2^n-2$个边缘分布。

可以看出，联合分布唯一决定了所有的边缘分布。 因为由上所述，想要知道哪个或哪些随机变量的分布，就将需要的随机变量留下来，不需要的随机变量写成$\infty$即可。但是，除了所有的随机变量都相互独立这种情况之外，我们应该牢记所有的边缘分布不能决定联合分布。因此，这就告诉了我们，有些情况，能研究联合分布就研究联合分布，而不能去研究边缘分布，因为得到了边缘分布，有可能得不到它的全体信息。

当然同一元的随机变量一样，随机向量也有离散型和连续型之分。

离散型随机向量

定义：设$\vec{X}=(X_1, X_2, \cdots,X_n)$是随机向量，如果每一个$X_i, 1 \leq i \leq n$都是离散型随机变量，则称$\vec{X}$是离散型随机向量。(最多取可数个值)

定义：若$\vec{X}$是一个离散型随机向量，则称$f_{X_1, X_2, \cdots,X_k}(x_1, x_2, \cdots,x_k)=P(X_1 = x_1, X_2 = x_2,\cdots,X_k = x_k)$为其联合密度函数

当然，当$x_1,x_2,\cdots,x_k$取遍所有可能的值时，有$\sum f_{X_1,X_2,\cdots,X_k}(x_1,x_2,\cdots,x_k)=1$

定义：如果$\vec{X}=(X_1,X_2,\cdots,X_k)$是一个随机向量，则对$X_1,X_2,\cdots,X_k$的任何一个子集合$X_{i_1},X_{i_2},\cdots,X_{i_m}$，称$f_{X_{i_1},X_{i_2},\cdots,X_{i_m}}(x_{i_1},x_{i_2},\cdots,x_{i_m})$叫做边缘密度函数

那么对于二维离散随机向量$(X,Y)$来说，$f_X(x), f_Y(x)$是边缘密度函数。取值为$x_i,y_i,i \ge 1$，称$f_{X,Y}(x_i,y_i)=P(X=x_i,Y=y_i)=p_{ij}$，为$X,Y$的联合密度函数，也叫联合分布。称$p_i=P(X=x_i)=\sum\limits_{j=1}^{\infty} P(X=x_i,Y=y_j)=\sum\limits_{j=1}^{\infty} p_{ij}$为$X$的边缘分布，称$q_j=P(Y=y_j)=\sum\limits_{i=1}^{\infty} P(X=x_i,Y=y_j)=\sum\limits_{i=1}^{\infty} p_{ij}$为$Y$的边缘分布。

多项分布

设实验$S$有$k$个可能的结果：$A_1,A_2,\cdots,A_k$，将$S$独立重复进行$n$次，用$X_i$表示$A_i, 1 \leq i \leq k$发生的次数，用$p_i$表示$A_i, 1 \leq i \leq k$发生的概率，则$\vec{X}=(X_1,X_2,\cdots,X_k)$具有联合分布：$f_{X_1,X_2,\cdots,X_k}(x_1,x_2,\cdots,x_k)=\frac{n!}{\prod\limits_{i=1}^{k}x_{i}!}\prod\limits_{i=1}^{k}p_i^{x_i}$，其中$x_i \ge 0,1 \leq i \leq k$，并且$\sum\limits_{i}x_i =n, \sum\limits_{i}p_i =1$，每次实验有且仅有一个$A_i$发生，称这种分布为多项分布。

这个分布很容易证明，就是从n个事件中选$x_1个A_1$事件发生，再从$n-x_1$个事件中选择$x_2个A_2$事件发生，以此类推最后一个事件发生了$n-x_1-x_2-\cdots-x_{k-1}$次，则就有了以上的公式。

在多项分布中，我们如何求每一个随机变量的边缘分布？

例如，在上述的多项分布中，求$X_1$的分布。

直观的来看，我们有$P(X_1=x_1)=\sum\limits_{x_2+x_3+\cdots+x_k=n-x_1}\frac{n!}{\prod\limits_{i=1}^{k}x_{i}!}\prod\limits_{i=1}^{k}p_i^{x_i}$，但是仔细思考一下就会发现，在一个实验中，$A_1$要么发生，要么不发生，且每一次发生的概率都是$p_1$，则我们可以看成服从$B(n,p_1)$的二项分布。同理，我们有$X_j \sim B(n,p_j)$，即$P(X_j=x_j)=C_n^{x_j}p_j^{x_j}(1-p)^{n-x_j}$

那么如何求$P(X_1=x_1,X_2=x_2)$

利用多项分布公式我们有:

$P(X_1=x_1,X_2=x_2)=\frac{n!}{x_{1}! x_{2}! (n-x_1-x_2)!}p_1^{x_1}p_2^{x_2}(1-p_1-p_2)^{n-x_1-x_2}$

连续型随机向量

先只考虑二维的情况。

定义：对于随机向量$(X,Y)$，如果有一个定义在$R^2$上的非负函数$f(x,y)$，对任何的$a<b,c<d$都有$P(a<X \leq b,c<Y \leq d)=\iint\limits_{D}f(x,y)dxdy$，其中$D=\{(x,y):a<X \leq b,c<Y \leq d\}$，则称$(X,Y)$为连续型随机向量，称$f(x,y)$为随机向量的联合密度函数，记作$(X,Y) \sim f(x,y)$。

当然对于这个定义，课本中也把它写作如下的形式，我们必须知道这两种是等价定义，就像对连续型随机变量的密度函数定义的那样。

定义：对于随机向量$(X,Y)$，如果有一个定义在$R^2$上的非负函数$f(x,y)$，有$F_{X,Y}(x,y)=\int_{-\infty}^x \int_{-\infty}^y f(u,v)dudv$，则称$(X,Y)$为连续型随机向量，并称$f(x,y)$为随机向量的联合密度函数。

注意，由于对于二重积分来说，在一个面积为0的区域上的积分仍然是0，因此，上述定义中的区域$D$可以是闭区间，也可以是开区间，这些都是等价的。上述等式的意义在于，如果在平面上有一个矩形区域，则随机向量落在这个矩形区域上的概率是以这个矩形区域为底，以随机向量的密度函数为顶的一个柱体的体积。可以证明对任何的子区域(不一定是矩形的)，都有$P((x,y) \in D)=\iint\limits_{D}f(x,y)dxdy=\iint f(x,y)I(D)dxdy$，其中$I(D)$为$D$的示性函数。

连续型随机向量的边缘分布与离散型随机向量边缘分布的定义是一样的。

如果$(X,Y) \sim f(x,y)$，则关于$X$的边缘密度函数是$f_X(x)=\int_{-\infty}^{\infty}f_{X,Y}(x,y)dy$，关于$Y$的边缘密度函数是$f_Y(y)=\int_{-\infty}^{\infty}f_{X,Y}(x,y)dx$。这两个公式是很好记的，要求$X$的密度函数，将$y$积掉即可，反之也是这样。

我们可以看出，如果$(X,Y)$是连续型随机向量，则$X$和$Y$都是连续型的随机变量，且它们的密度函数由联合密度函数唯一决定。但是反之，如果对于连续型随机变量$X$和$Y$，它们都有密度函数，我们能不能找出一个连续型随机变量$(X,Y)$的联合密度函数呢。
