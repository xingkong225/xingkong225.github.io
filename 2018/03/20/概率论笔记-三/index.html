概率论笔记(三)

发表于 2018-03-20 | 分类于 专业课
字数统计: 4,233 | 阅读时长 ≈ 17

写了这么多的理论知识，有必要将学过的各种分布函数系统地总结一下，并在总结的过程中回顾以前的知识，那么，我们开始吧。说明一点，以下我会用$f_X(x)$或者$p(x)$来表示随机变量的密度函数；使用$F_X(x)$或者$F(x)$来表示随机变量的分布函数。

我们知道随机变量有离散型随机变量和连续型随机变量之分，因此分开来说明是有必要的。

离散型随机变量的分布函数

一、两点分布(又称伯努利分布)

定义：如果随机变量$X$的可能值是0和1且概率分布为$P(X=1)=p,P(X=0)=1-p(0 \leq p \leq 1)$，则称$X$服从两点分布，也叫伯努利分布。

通常，我们也会将$1-p$写作$q$。有时，两点分布也会写成更加复杂也显得比较专业的形式，但是我们必须要知道以下的两种写法是与上方的定义是等价的。

如果$X$满足两点分布(伯努利分布)，我们有$E(X)=p,var(X)=pq,m(t)=pe^t+q$(还记得吗，$m(t)$是矩函数，如果忘记了可以去看看我的上一个笔记内容)

二、二项分布

定义：设随机变量$X$的所有可能取值是$0,1,\cdots,n$，且$P(X=k)=C_n^k p^k (1-p)^{n-k}(k=0,1,\cdots,n)$，这里$n \ge 1,0 \leq p \leq 1$，则称$X$服从参数为$n,p$的二项分布，记做$X \sim B(n,p)$。

二项分布的背景是，若在每次试验中事件$A$发生的概率都是$p$，则在$n$次独立重复实验中，$A$发生$k$次的概率即是上述的二项分布。并且我们应该能看得出来，当$n$的值是1时，二项分布就是两点分布。

由于二项分布中每一次的事件都是独立的，因此在每一次的实验中，成功的概率是$p$，失败的概率是$1-p$，其实都是两点分布，也就是伯努利分布。

如果随机变量$X$满足二项分布，则有$E(X)=np,var(X)=npq,m(t)=(q+pe^t)^n$。在这里计算的时候利用矩函数会更加方便一些。

这里也给出二项分布的另几种表达方式。

定理：令$p_n(k)=C_n^k p^k (1-p)^{n-k}$，则当$(n+1)p$不是整数时，$k < (n+1)p$时，$p_n(k)$单调上升，当$k > (n+1)p$时，$p_n(k)$单调下降。当$(n+1)p$是整数时，$p_n((n+1)p)=p_n((n+1)p-1)$，同样在其左侧单调上升，在其右侧单调下降。

据此，我们知道，$p_n(k)$作为$k$的函数先增后减，在$[(n+1)p]$处取得最大值，并且当$(n+1)p$是整数时，有两个最大值点，即$(n+1)p=(n+1)p-1$.

三、超几何分布

定义：若随机变量$X$的分布满足，$P(X=k)=\frac{C_D^{k}C_{N-D}^{n-k}}{C_N^n}$，则称$X$服从超几何分布。

我们可以将公式这样理解，在共有$N$个产品中，一共有$D$个不合格的产品，现从中取出$n$个，则这$n$个产品中所含的不合格的产品数则是上述的离散型随机变量。

超几何分布与二项分布有着密切的关系：

定理：设在超几何分布中$D$是$N$的函数，即$D=D(N)$，并且有$\lim\limits_{N \rightarrow \infty} \frac{D(N)}{N}=p(0 < p < 1)$，则有$\lim\limits_{N \rightarrow \infty}\frac{C_{D(N)}^{k}C_{N-D(N)}^{n-k}}{C_N^n}=C_n^k p^k(1-p)^{n-k}$，对任何固定的$k \ge 0$

上述定理的直观意义是，当一批产品的总量$N$很大时，而不合格产品所占的比例是$p$，则从整批中随机抽取$n$个时所抽到的不合格个数近似服从参数是$n,p$的二项分布。

对于超几何分布，其期望$E(X)=n·\frac{D}{N}$，方差$var(X)=n·\frac{D}{N}·\frac{N-D}{N}·\frac{N-n}{N-1}$。

四、离散均匀分布

定义：设随机变量$X$的取值范围是$(1,2,\cdots,N)$($N$是大于1的整数)，且$P(X=k)=\frac{1}{N}(k=1,2,\cdots,N)$，则称$X$服从离散均匀分布。

当然，我还会将离散均匀分布的另一种表达方式写在下面：

对于离散均匀分布，我们有$E(X)=\frac{N+1}{2},var(X)=\frac{N^2-1}{12},m(t)=\sum\limits_{j=1}^{N}e^{jt}\frac{1}{N}$。

离散均匀分布在抽签，抽样调查及彩票中奖等方面的研究中常遇到，有时参数$N$是未知的，你需要去估计它。

五、几何分布

定义：设$X$是一个随机变量，且值为正整数，若满足$P(X=k)=(1-p)^{k-1}p(k=1,2,\cdots)$，其中$(0 < p < 1)$，则称$X$服从几何分布。

如果我们令$p_k=(1-p)^{k-1}p(k=1,2,\cdots)$，则$p_k$构成了一个以公比为$1-p$的几何级数，”几何分布”的名称就来源于此。由此可见，几何分布的众数必然为0！

我们可以这样理解几何分布，设某射击选手向某目标连续射击，直到击中为止，若他单次射击射中目标的概率是$p$，则他首次射中目标所需要的射击次数就是一个随机变量，且显然服从几何分布。

对于几何分布，我们有$E(X)=\frac{q}{p},var(X)=\frac{q}{p^2},m(t)=\frac{p}{1-qe^t}$，其中$q=1-p$

几何分布还有着一个十分有趣的性质(无后效性)：如果$X$是一个几何分布，那么就有$P(X \ge i+j | X \ge i)=P(X \ge j),i,j=0,1,2,\cdots$，也就是如果在前$i$次实验都没有成功的条件下，再至少进行$j$次实验才能成功的概率，与在没有任何条件下，至少经过$j$次实验才能成功的概率是相等的。换言之，若进行了$i$次实验都没有成功，那么需要至少再做$j$次实验才能成功的概率，不依赖于以前的实验，也就是说，将过去的记忆忘记了，就像实验重新开始一样，这种性质也叫作无后效性或者无记忆性。

六、负二项分布

定义：设随机变量$X$的取值范围是$(r,r+1,\cdots)$，(r是正整数)，且$P(X=k)=C_{k-1}^{r-1}p^r(1-p)^{k-r}$，则称$X$满足负二项分布。

有些课本还会使用下方的表达式表示负二项分布，但是它们表示的意义是一样的，可以互相推导：

在这里，如果$r=1$，则负二项分布就变成了几何分布。

负二项分布的背景是，设某一个射击选手向一个目标连续进行射击，决心射中$r$次后停止，若他单次射击射中目标的概率是$0 < p < 1$，则他需要射击的次数就是一个随机变量，显然满足负二项分布的表达式。负二项分布的特点是最后一次发生的事件一定是成功的。

我们可以看出来，就像二项分布与伯努利分布的关系一样，负二项分布也是可以由若干个几何分布”叠加形成”。

对于负二项分布，我们有$E(X)=\frac{rq}{p},var(X)=\frac{rq}{p^2},m(t)=\left[\frac{p}{1-qe^t}\right]^r$，其中$q=1-p$，同样的我们使用矩函数会使我们的运算更加的方便。注意这里的期望和方差都与几何分布相应的期望和方差相差一个$r$。

七、泊松分布

定义：设随机变量$X$所有可能值是全体的非负整数，且$P(X=k)=\frac{1}{k!}\lambda^k e^{-\lambda}(k=0,1,2,\cdots)$，其中$\lambda$是整数，则称$X$是服从参数为$\lambda$的泊松分布。

泊松在1837年首先提出了这个分布，他把这个分布看作是二项分布$B(n,p)$当$np \rightarrow \lambda$时的极限。当二项分布的$n$很大而$p$很小时，泊松分布可以作为二项分布的近似，后来人们发现了这个分布可以用来刻画许多的随机现象。在之后我们会看到有连续型随机变量的指数分布，它其实就可以由泊松分布推导出来。

泊松分布的参数$\lambda$是单位时间或者单位面积内随机事件的平均发生率。它适合于描述单位时间内随机事件发生的次数，为许多的随机现象提供了一个现实的模型。就像课本中所描述的那样，发生的事情可能是致命的交通事故、粒子的发射、电话的到来、陨星的撞击、材料的缺陷、电线的损坏等等随机事件，如果对观察中的现象的某些假设得到了满足，则泊松模型就是正确的模型。值得注意的是，泊松分布中随机变量的取值只能是非负的整数，而且参数$\lambda$必须要大于零。

对于泊松分布，我们有$E(X)=\lambda,var(X)=\lambda,m(t)=e^{\lambda(e^t-1)}$

这个公式是很重要的，可以用来计算概率：$P[Z(t)=z]=\frac{e^{-vt}(vt)^z}{z!}$，其中$z=0,1,2,\cdots$，$t$为一段时间，并且有$\lambda=vt$，$v$即事件发生的频率。

例如：如果一个公司在一分钟内平均收到四个电话，那么在接下来的三分钟，一个电话都收不到的概率是多少？$P[Z(3)=0]=\frac{e^{-3 \times 4} {3 \times 4}^0}{0!}$.接下来一分钟，至少收到两个电话的概率是多少？$P[Z(1) \ge 2]=1-P[Z(1)=0]-P[Z(1)=1]$，直接带入公式即可求得。

我们需要思考这样一个问题，设$X$服从泊松分布，则当$k$取何值时，其概率最大？那么请看下面的一个定理。

设$X$满足泊松分布，则当$k < \lambda$时，密度函数单调上升；当$k > \lambda$时，密度函数单调下降；当$\lambda$是一个整数并且$k=\lambda$时，有$\frac{e^{-\lambda}\lambda^{k-1}}{(k-1)!} = \frac{e^{-\lambda}\lambda^k}{k!}$，其中$k$是随机变量，取值为$(0,1,2,\cdots),\lambda$为参数。

因此，我们可以得到结论，在频率的附近，事件发生的概率最高，然后两边就开始下降，当随机变量变得过大或者过小都不太可能。比如平均一分钟收到四个电话，这个是最有可能的，收到的电话更多或者更少都不太可能。

连续型随机变量的分布函数

一、均匀分布

定义：设随机变量$X$满足$\begin{eqnarray}f_X(x)=\begin{cases}\frac{1}{b-a} &a \leq x \leq b\\0 &otherwise\end{cases}\end{eqnarray}$，则称$X$服从区间$[a,b]$上的均匀分布。

我们可以看到，$P(c<X<d)=\frac{(d-c)}{(b-a)}$，这表明，$X$取值于$[a,b]$中任何小区间的概率与该小区间的长度成正比，而与该小区间的位置无关，这就是均匀分布的概率意义。例如去公共汽车站候车的时间就是均匀分布，因为出行者到达车站的时刻是随机的。

对于均匀分布，我们有$E(X)=\frac{a+b}{2},var(X)=\frac{(b-a)^2}{12},m(t)=\frac{e^{bt}-e^{at}}{(b-a)t}$

同时也不难求出其分布函数为$F_X(x)=\left(\frac{x-a}{b-a}\right)I_{[a,b]}(x) + I_{(b,\infty)}(x)$

值得注意的是，在定义均匀分布时，我们指定小区间为一个闭区间，但是仍然可以定义在开区间以及半开半闭区间上的均匀分布，只不过这个时候的分布函数与原来是一样的，但是密度函数不一样，因为显然区间发生了变化。我们已经知道，改变区间内有限个点不会影响在该区间上积分的值，所以即使密度函数不一样，但是分布仍然是一样的。

二、正态分布

定义：如果随机变量的密度函数为

其中$\mu$是实数，$\sigma$是正数，则称$X$服从参数为$\mu, \sigma$的正态分布，记为$X \sim N(\mu, \sigma)$，正态分布也叫做高斯分布。

密度函数在直角坐标中的图像为一个钟形，最大值点是$x=\mu$，图像关于直线$x=\mu$对称，在$x=\mu+\sigma$和$x=\mu-\sigma$处有拐点，当$x \rightarrow \infty$和$x \rightarrow -\infty$时曲线均以$x$轴为渐近线。并且当$\sigma$大时曲线平缓，当$\sigma$小时曲线陡峭。

我们使用$\mu, \sigma$这些符号的原因是由于，已经被证明其中$\mu$就是均值，$\sigma^2$就是方差。

当$\mu=0,\sigma^2=1$时的分布叫做标准正态分布，其密度函数是$f_X(x)=\frac{1}{\sqrt{2\pi}}e^{\frac{-x^2}{2}}$，我们可以经过$y=\frac{x-\mu}{\sigma}$这种变换将正态分布转换成标准正态分布，并且已经证明了$A=\frac{1}{\sqrt{2\pi}}\int_{-\infty}^{\infty}e^{\frac{-x^2}{2}} dx$的值为1，其中需要用到数学分析的知识，即将重积分化为累次积分，这个证明方法也是很巧妙地，读者应该会自己推导。

通常的，我们使用$\phi(x)=\frac{1}{\sqrt{2\pi}}e^{\frac{-x^2}{2}},\Phi(x)=\int_{-\infty}^{x}\phi(u) du$这种记法来表示标准正态密度及其分布函数。$\Phi(x)$有着一个十分不错的性质，即$\Phi(-x)=1-\Phi(x)$，我们应该知道，标准正态分布是一个很重要的概念，几乎所有的正态分布都可以由标准正态分布函数也就是$\Phi(x)=\int_{-\infty}^{x}\phi(u) du$计算出来(可见下面的定理)。关于标准正态分布还有一个表格，因为我们几乎不可能将对所有的$\Phi(x)=\int_{-\infty}^{x}\phi(u) du$积分值算出来，查阅表格是一个比较方便的事情。

下面的定理表明，我们可以将任何正态分布的随机变量落在某一个区间上的概率转换到标准正态分布上。

定理：如果$X \sim N(\mu,\sigma^2)$，则有$P(a < X < b)=\Phi(\frac{b-\mu}{\sigma}) - \Phi(\frac{a-\mu}{\sigma})$

推论：设$X \sim N(\mu,\sigma^2)$，则对一切正数$k$，有$P(\mu-k\sigma < X < \mu+k\sigma)=2\Phi(k)-1$，这个推论可以从上述定理推导出来。

从上述推论以及参阅标准正态分布表，我们可以知道，

$P(\mu-\sigma < X < \mu+\sigma) = 0.6826$

$P(\mu-2\sigma < X < \mu+2\sigma) = 0.9546$

$P(\mu-3\sigma < X < \mu+3\sigma) = 0.9974$

$P(\mu-6\sigma < X < \mu+6\sigma) = 0.\underbrace{9\cdots9}_{9个9}0134$

由此可见，正态随机变量$X$的取值基本落在区间$(\mu-3\sigma,\mu+3\sigma)$内，落在$(\mu-6\sigma,\mu+6\sigma)$之外的情形极为罕见。

人们发现，许多生理指标，例如身高、体重、血压等，以及产品的质量指标，例如硬度、抗压强度等，都服从或者近似服从正态分布，正态随机变量的大量出现不是偶然的，以后关于”中心极限定理”就可以从理论上解释这一点。

附标准正态分布表格(来源百度图片)其中$\Phi(x)=\int_{-\infty}^{x}\phi(u) du$，例如要想知道$\Phi(2.45)$，则查找表格，有$\Phi(2.45)=0.9929$

三、指数分布

定义：如果随机变量的密度函数满足$\begin{eqnarray}f_X(x)=\begin{cases}\lambda e^{-\lambda x} &x > 0\\0 &x \leq 0\end{cases}\end{eqnarray}$，其中$\lambda > 0$，则称$X$服从参数为$\lambda$的指数分布。

若$X$服从指数分布，则$P(a < X < b)=\lambda\int_a^b e^{-\lambda x} dx=e^{-\lambda a}-e^{-\lambda b},P(X > a)=e^{-\lambda a},P(X > 0)=1$。

指数分布也具有无记忆性，看下述的定理：

定理：设$X$是取值为非负的随机变量，则对任何$s \ge 0,t \ge 0$，都有$P(X > s+t | X > s)=P(X > t)$恒成立的充分必要条件是$X$服从指数分布

注意这个定理是一个充分必要条件，现只以充分性做以证明。

证明：($\Leftarrow$)$P(X > s+t | X > s)=\frac{P(X > (s+t))}{P(X > s)}=\frac{e^{-\lambda (s+t)}}{e^{-\lambda s}}=e^{-\lambda t}=P(X > t)$

如果$X$满足指数分布，则有$E(X)=\frac{1}{\lambda},var(X)=\frac{1}{\lambda^2},m(t)=\frac{\lambda}{\lambda-t}$。
